# Vibe Check Machine 🎯

A simple and fun web app that asks a few questions and tells you your "vibe" based on your answers.

## 🚀 How to Run

1. Download or clone this project  
2. Open the folder  
3. Double-click `index.html` to run in your browser  

💡 Tip: You can also use VS Code Live Server for a better experience.
